Insurance

Getting Insurance

The easiest way to get Insurance is to use Ruby Gems: 

$ gem install insurance

Setting up your Rails application

Use the railscov command to install a Rake task into your Rails application directory. 

$ railscov -A MyRailsApp

Running Insurance on your Rails application

Use the installed Rake task. 

$ rake insruance

This will generate an insurance/ directory with an HTML file for each file in your project. Lines of code that were not executed will be marked with a red tinted background.
